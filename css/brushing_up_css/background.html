Brushing up Background Images

The following are the properties that are used to manipulate backgrounds
  • background-color : Sets the color of the background of the element
  • background-image: Sets an image using url() as background of an element
  • background-repeat: Naturally, the background repeats itself to occupy the entire screen, this can be changed using this property, it has 4 values: repeat-x,repeat-y,repeat and no-repeat.
  • background-size: This sets the size of the background: it has 4 values: cover (covers the entire page), contain(it the background area entirely while maintaining aspect-ratio), auto(original size,default), custom width and height
  • background-position: This can be set to top, bottom, left, right, center and their combinations
  • background-attachment: This can be set to fixed(fixed on the page) or scroll(scrolls with the page)
