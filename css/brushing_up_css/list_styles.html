Brushing up List Styles

There are 3 css properties which one can use to change the styles of list, they are:

  • list-style-type: Used to change the style of list markers.
    • Hello World
    • This is ul1, it uses disc type bullets.
    • Hello World
    • This is ul2, it uses circle type bullets.
    • Hello World
    • This is ul3, it uses square type bullets.
    • Hello World
    • This is ul4, it doesn't use bullets.

    Other than these bullet types several numeric type list markers are also available, both in uppercase and lowercase

  • list-style-position: It determines where the list markers (bullets or numbers) are placed in relation to the content. It has two values, inside and outside
    • This is a list element
    • list-style-position:inside
    • This is a list element
    • list-style-position:outside
    list-style-image: This property can be used to change the list markers to an image.
