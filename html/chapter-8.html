This is a section

The section element in HTML is a semantic element used to define a standalone section within a document, typically with a heading. It groups related content together under a common theme, like a chapter, introduction, or a specific section of a report

This is an article

The article element in HTML represents a self-contained composition in a document, page, application, or site, which is intended to be independently distributable or reusable.
